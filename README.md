This project allows a user to generate a fake dataset with no coding experience required.
<img width="1316" alt="Screenshot 2024-10-02 at 1 36 41 AM" src="https://github.com/user-attachments/assets/66c9767d-3535-4a86-9f3d-68d90ad36a52">

Users can enter a schema idea to get started with some help from chatGPT. 
<img width="1243" alt="Screenshot 2024-10-02 at 1 55 52 AM" src="https://github.com/user-attachments/assets/9bf7f643-0db1-4a22-a2a4-5b63fa3ed167">


To generate a fake dataset, they can enter in a table name and add columns. The type of data can be selected from a dropdown menu, which generates fake data of the specified type with the help of [Link Text](https://www.mockaroo.com/docs)
<img width="1243" alt="Screenshot 2024-10-02 at 1 57 15 AM" src="https://github.com/user-attachments/assets/c0bd28ac-9714-48a2-93d2-433dc35b7358">

Users can add additional data and download their dataset as a CSV.
<img width="1249" alt="Screenshot 2024-10-02 at 1 57 38 AM" src="https://github.com/user-attachments/assets/a79c31a8-7866-4895-b5d9-835b1827c79a">

The table will be saved if they want to revisit their work.
